# Project Writing Guidelines

## Language

- Write in **English**.
- Use **simple, clear, and technical language**.
- Write in **third person** throughout the report.
- Ensure:
  - Correct spelling
  - Proper punctuation
  - Correct capitalization
  - Appropriate use of italics
  - Good sentence structure

---

# Paper Specifications

- **Paper Size:** A4 (21.0 cm × 29.7 cm)
- **Print:** One side of the paper only
- **Length:** 30–50 pages *(excluding preliminary pages and appendices)*

---

# Page Margins

| Margin | Size |
|--------|------|
| Left | 3.0 cm |
| Top | 2.54 cm |
| Right | 2.54 cm |
| Bottom | 2.54 cm |

---

# Font

- **Font Family:** Times New Roman
- **Font Size:** 12 pt
- **Font Color:** Black, for all text

**Exceptions:**
- Smaller font may be used in tables, figures, and appendices if legible.
- Larger font may be used on the cover page and title page.

---

# Line Spacing

- Double spacing throughout the report.
- Single spacing inside tables.
- 1.5 spacing for headings and subheadings.

---

# Paragraph Style

Choose **one** style and use it consistently throughout:

- Indented paragraphs, or
- Block paragraphs

---

# Headings

Use a maximum of **three heading levels**.

## Level 1 (Chapter Title)

- ALL CAPITAL LETTERS
- Bold
- Numbered by chapter

Example:

```text
CHAPTER ONE
INTRODUCTION
```

## Level 2

- Bold
- Title Case

Example:

```text
1.1 Background of the Study
```

## Level 3

- Italicized
- Title Case

Example:

```text
1.1.1 *Problem Definition*
```

### Additional Rules

- Left-align all headings.
- Number headings consecutively using Arabic numerals.
- Leave **one blank line** between a heading and the preceding paragraph, table, or figure.
- Do **not** leave a blank line between consecutive headings.
- Never leave a heading as the last line of a page.

---

# Tables and Figures

## Numbering

Use chapter-based numbering.

Examples:

- Table 3.1
- Table 3.2
- Fig. 4.1
- Fig. 4.2

## Placement

- Table titles go **above** the table.
- Figure captions go **below** the figure.
- Keep each table or figure on the same page as its title.
- If applicable, state the source immediately below the table or figure in *italics*.

---

# Equations

- Use an equation editor (e.g., Microsoft Equation Editor, MathType).
- Center all equations.
- Number equations consecutively using Arabic numerals.

Example:

```text
(3.1)
```

Do **not** write the word **Equation** before the number.

---

# Pagination

## Preliminary Pages

Use **Roman numerals**:

```text
i, ii, iii, iv, ...
```

## Main Report

Use **Arabic numerals**:

```text
1, 2, 3, ...
```

Place page numbers at the **bottom center** of each page.

---

# References

- Use **APA referencing style**.
- Arrange references alphabetically by the first author's surname.
- Every in-text citation must appear in the reference list.
- Every reference listed must be cited in the text.
- Do **not** include citations in the abstract.

Acceptable sources include:

- Books
- Journal articles
- Conference proceedings
- Reports
- Websites

Do **not** cite dictionaries, encyclopedias, or generic homepages (e.g., Google).

---

# General Writing Tips

- Maintain consistency in formatting throughout the document.
- Ensure all chapters begin on a new page.
- Use automatic table of contents generation in Microsoft Word or LaTeX.
- Proofread carefully before submission to eliminate grammatical and formatting errors.
